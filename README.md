# Tabuada-Java-Forms
Este projeto consiste em uma aplicação desenvolvida em Java Forms (NetBeans) que permite ao usuário gerar a tabuada de um número de forma simples e interativa através de uma interface gráfica.

O usuário informa o valor desejado e o sistema exibe automaticamente os resultados da tabuada na tela, facilitando a visualização e o aprendizado.

O projeto foi criado com o objetivo de praticar lógica de programação, uso de eventos e desenvolvimento de interfaces gráficas em Java.
